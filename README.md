### 五、项目介绍
项目编号：python076

学生会网站利用计算机网络实现信息化管理，使整个学生会网站的发展和服务水平有显著提升。
系统拥有管理员和用户两个角色，主要具备注册登录、个人信息修改、用户、校内报道、部门介绍、品牌活动、问答信息、问答类型、互动交流等功能模块。将纸质管理有效实现为在线管理，极大提高工作效率。
